Este repositório contém o meu projeto de portfólio desenvolvido no âmbito da disciplina.
O objetivo principal foi criar um site simples e funcional que apresentasse o meu percurso, competências, projetos e certificações, aplicando os conhecimentos adquiridos ao longo do módulo.
🎯 Objetivos do Trabalho

    Criar um site estático usando HTML, CSS e JavaScript

    Organizar o conteúdo em várias páginas

    Aplicar princípios de responsividade

    Utilizar animações simples (Scroll Reveal)

    Estruturar o projeto de forma limpa e organizada

    Integrar ficheiros externos (imagens, PDFs, CSS, JS)

🧱 Estrutura do Projeto
Code

brunodinizprojecto604/
│
├── assets/
│   ├── certificacoes/   → PDFs das certificações
│   └── img/             → Imagens usadas no site
│
├── css/                 → Ficheiros de estilos
│
├── js/                  → JavaScript (scroll reveal, interações)
│
├── pages/               → Páginas internas (Sobre, Percurso, Tecnologias, Projetos)
│
└── index.html           → Página inicial

🛠️ Tecnologias Utilizadas

    HTML5 — estrutura das páginas

    CSS3 — estilos, layout e responsividade

    JavaScript — animações e interações

    Git & GitHub — controlo de versões e alojamento do código
