Principais Tecnologias aprendidas até agora

HTML: A base de toda a web, permitindo estruturar o conteúdo de forma semântica e acessível.
CSS: Essencial para estilizar e tornar as páginas visualmente atraentes, com habilidades em Flexbox e Grid para layouts responsivos.
JavaScript: A linguagem de programação que traz interatividade às páginas, com conhecimentos em manipulação do DOM, eventos e lógica de programação.
React: Biblioteca JavaScript para construção de interfaces de usuário, com conhecimento em componentes, hooks e gerenciamento de estado.
